# Ideas para sumar a la invitacion

Estas son opciones posibles para futuras iteraciones. La idea no es agregar todo, sino elegir lo que aporte sin volver la tarjeta pesada o demasiado cargada.

## Muy utiles

- RSVP con Google Forms, Tally o Typeform.
- Boton para agregar el evento al calendario.
- Links reales de Google Maps para ceremonia y fiesta.
- Boton para copiar alias o datos bancarios.
- Mensaje de menu especial: vegetariano, celiaco, alergias, etc.
- Galeria breve con 4 a 6 fotos.

## Lindas pero opcionales

- Seccion de canciones para sugerir musica.
- Hashtag o link a Instagram.
- Timeline simple de la noche: ceremonia, recepcion, cena, baile.
- Frase corta de apertura, sobria y no cursi.
- Modo "compartir invitacion" con Web Share API en celular.
- Animaciones suaves al scrollear, similares a la referencia.

## Mas avanzadas

- Confirmaciones guardadas en Supabase.
- Panel privado para ver invitados confirmados.
- Lista de regalos editable.
- Dominios personalizados, por ejemplo `luchiyfacu.com`.
- Version con clave para invitados.

## Recomendacion

Para la proxima etapa conviene priorizar:

1. Datos reales de ceremonia y fiesta.
2. RSVP externo.
3. Galeria corta.
4. Boton de calendario.
5. Pulido visual con fotos definitivas.
